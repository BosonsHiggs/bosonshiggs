Commands of discord

Commands for Users

Any user can use these commands

Commands What does it do?
?help ,?ajuda or ?hp Return a list of available commands.
?eu Return your user information.
?sobre @*user* Return information from other users .
?myinvite or ?miv #channel-name Creates a link to the channel by the informed ID If ID is empty, creates a link to the current channel.
?ping Returns the average bot response time.
?mat, ?math or ?m mathematical-operation Returns the result of a mathematical operation.
?arcade Checks if there is any beta game.
?dicio, ?dic or ?dc "word" Search for the definition or classification of a word in Portuguese [br] or English [us].
?translate or ?traduzir or ?trans language text Translate a word into Portuguese[br].
?sortnum None interval ?sortenum None Draws a number between a range (eg. sortnum 1-10).
?sort None, ?sorte None, ?sortear None, ?sorteio None, ?escolher None, ?choose None or ?pick None Draw a word among others. Don't forget to separate with a comma! (ex. sort grape, pear, apple).
?staffajuda "text" or ?sta "text" Send a message to the staff channel.
?meajude or ?ma Help the bot's creators.
?botinvite @bot-name or ?bi @bot-name Creates an invitation link to another bot.

Commands for staff

Staffers must have permission to "channel manager"

Commands What does it do?
?clear number Delete a certain number of messages.
?clear_all Delete all messages from a channel.
?invites or ?iv Lists user invites.
?lock @role #channel-name Locks a channel, except for "@role" If "#channel-name" is not informed, the current channel will be blocked You can also use only ?lock to lock for everyone.
?unlock @role #channel-name Unlocks a channel, except for "@role" If "#channel-name" is not informed, the current channel will be blocked You can also use only ?unlock to unlock for everyone.
?defstaff #channel-name or ?dfc #channel-name Defines a channel for users to request help from the staff.
?udefstaff #channel-name or ?udfc #channel-name Removes the staff help channel.
?defin #channel-name or ?dfi #channel-name Set the user input channel.
?udefin #channel-name or ?udi #channel-name Defines the channel as not being user input.
?defout #channel-name or ?dfo #channel-name Sets the user output channel.
?udefout #channel-name or ?udo #channel-name Defines the channel as not being an user output.
?defban #channel-name Sets a channel to display banned users.
?udefban #channel-name Defines the channel as not being banned users.
?channelvname #channel-name "new-name" or ?chvn #channel-name "new-name" Renames a voice channel to "{'Sets the user output channel'}".
?channeltname #channel-name "new-name" or ?chtn #channel-name "new-name" Renames a text channel to "{'Sets the user output channel'}".
?statuschannel or ?sch Creates status channels, which show the number of users and bots on the server.
?udefstatus or ?uds Removes status channels.
?sorterole @cargo or ?sr *@cargo* Randomly selects a user who has a specific role.
?addquiz question right_answer option1 option2... Adds questions to the game quiz.
?delquiz or ?dqz Delete a question or the entire quiz. Use the commands mentioned to see the examples!

Commands for admins

Users must have permission from "Administrator"

Commands What does it do?
?allmembers or ?am Displays the number of users and bots on the server.
?ban @user or ?ban @user "reason" Ban a user.
?unban ID-user Unban a user.
?kick @user or ?kick @user "reason" Kick a user.
?mkch "category" channel-name Creates a normal text channel.
?mkchv "category" channel-name Creates a normal voice channel.
?mkpch "category" channel-name Creates a secret text channel.
?mkpchv "category" channel-name Creates a secret voice channel.
?delch "category" DDeletes a category and all existing channels in the same.
?remove_roles @user @role or ?rmr @user @role Removes an @{'role'} role from the user @user.
?setprefix prefix Changes the prefix of the bot (ex. ?setprefix p!)
?remind DD/MM/YYYY @user @role or ?rd DD/MM/YYYY @user @role Adds a temporary role to the user until the date DD/MM/YYYY
?addroles @oldrole @newrole or ?adr @oldrole @newrole Adds an @{'newrole'} role to users who have an @{'oldrole'} role.
?adduser_roles @user @role or ?adur @user @role Adds an @{'role'} role to the @user user.
?wellcome channel-name or ?wcm channel-name Defines a channel as a record by reaction. You must create a channel just for that and a role with the name "Membro" or "Member".
?udefwellcome or ?udfw Removes the bot as an authentication method on the welcome channel.
?reactionrole @role emoji or ?rr @role emoji Reaction roles. You can enter up to 10 roles at a time.
;roletime time @role emoji or ?rt life @role emoji ReOffers a position for a period of time in seconds. (e.g. ;rt 60 @role emoji).
?defchatguild channel-name or ?dfcg channel-name Defines a channel as chatguild. With this channel, owners and admins will be able to communicate with other owners and admins.
?udefchatguild channel-name or ?udcg channel-name Defines a channel as not being a chatguild.
?listguild or ?lgd Displays the possible chatguild servers.
?chatguild ID-server "Message" or ?cgd ID-server "Message" Sends a message to the server informed by the ID.
